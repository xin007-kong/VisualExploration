# 括号匹配工具

这是一个使用栈数据结构实现的括号匹配检验工具。

## 功能

- 输入包含括号的字符串。
- 检查括号是否正确匹配。
- 可视化栈的状态，帮助学生理解括号匹配的过程。

## 使用方法

1. 在 Streamlit 应用中输入包含括号的字符串。
2. 点击"检验"按钮。
3. 观察栈的状态变化和匹配结果。

## 为什么使用栈?

栈是一个后进先出的数据结构，非常适合处理括号匹配问题。当我们遇到一个开括号时，我们将其压入栈中。当我们遇到一个闭括号时，我们检查它是否与栈顶的开括号匹配。这种方法简单且有效。

## 测试用例

1.**正确匹配的用例** ：

- `()`
- `[]`
- `{}`
- `()[]{}`
- `{[()]}`

  2.**不匹配的用例** ：

- `(`
- `]`
- `([)]`
- `{{[[(())]]}}}`
- `[{]}`

  3.**复杂的括号组合** ：

- `(([{()}]))`
- `[{()}([])]`
- `{{[[(())]]}}`
